# 내가 쓸 수 있는 도구 목록 — 조문지

작성일: 2026-04-09

## Claude Code 기본 도구

| 도구 | 역할 | 활용 예시 (R&D 신사업전략팀) |
|------|------|--------------------------|
| **Read** | 파일 읽기 | PPR 보고 파일 내용 확인 |
| **Write** | 파일 만들기 | 연구개발비 분석 결과 파일 생성 |
| **Edit** | 파일 수정 | 보고서 초안 수정 |
| **Bash** | 터미널 명령 실행 | git 저장/올리기 |
| **Glob** | 파일 검색 | Excel 예산 파일 찾기 |
| **Grep** | 내용 검색 | 보고서에서 특정 항목 찾기 |
| **WebSearch** | 웹 검색 | 최신 기술 트렌드 조사 |
| **WebFetch** | 웹페이지 읽기 | 외부 자료 가져오기 |
| **TodoWrite** | 할 일 관리 | 분석 작업 단계 추적 |
| **Agent** | 하위 AI 실행 | 복잡한 연구 기획 작업 위임 |
