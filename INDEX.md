## 바이브 코딩 (Vibe Coding) — MW5 핵심 개념

코드를 직접 작성하지 않고 **AI에게 자연어로 설명**해서 원하는 결과를 만드는 방식.

| 기존 방식 | 바이브 코딩 |
|----------|-----------|
| 엑셀 함수 직접 작성 | "매출 상위 5개 제품을 찾아서 표로 정리해줘" |
| Python 코드 작성 | "이 데이터에서 불량 패턴을 찾아줘" |
| 보고서 양식 수동 작성 | "팀장님 보고용으로 바꿔줘" |

→ [[MW5-워크플로우와-스킬]] + [[Concept-프롬프트]] 4법칙이 핵심
