## MW6 — 옵시디언 + 통합 지식 워크플로우 (정리 → 연결 → 검색)

### 핵심 개념

| 개념 | 한 줄 설명 |
|------|-----------|
| **옵시디언 (Obsidian)** | .md 파일 기반 지식 관리 앱. 내 PC에 저장 |
| **Vault** | 옵시디언의 저장소. 모든 노트가 들어있는 서랍장 |

### 3가지 vault 정리법 비교

| 방식 | 만든 사람 | 핵심 원리 | 적합한 사람 |
|------|----------|----------|-----------|
| 태그+링크 | Steph Ango (옵시디언 CEO) | 폴더 없이 wikilink로 연결 | 정리 귀찮은 분 |
| Mine vs Library | 김재경 (강사) | 내 것 vs 외부 것 구분 | 콘텐츠 생산자 |
| AI 최적화 위키 | Karpathy (테슬라 AI) | AI가 읽기 쉬운 메타 구조 | AI 파워유저 |

### 내가 선택한 구조

```
vault/
├── Mine/연구개발비/     ← 월별 예산 실적 보고서
├── Mine/PPR/           ← PPR 보고 파일
├── Mine/기획/          ← 연구 기획서
├── Library/타부서-참고자료/  ← 타 부서에서 받은 자료
└── Archive/            ← 완료된 자료 보관
```

> 선택 이유: "종류 먼저, 월별 비교 가능하게"

### Claude Code vs ChatGPT/Gemini 핵심 차이 4가지

| | Claude Code + 옵시디언 | ChatGPT / Gemini |
|---|---|---|
| **저장** | 내 vault에 .md로 영구 저장 | 대화창에만 남음 |
| **연결** | wikilink로 기존 자료와 자동 연결 | 각 대화가 고립 |
| **출처** | 노트 상단에 URL+날짜 자동 기록 | 추적 어려움 |
| **맥락** | student-profile.md로 자동 반영 | 매번 다시 설명 |
