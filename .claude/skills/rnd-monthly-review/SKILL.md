# R&D 월간 실적 분석 스킬

## 역할
조문지님의 R&D 신사업전략팀 월간 연구개발비 실적 분석 루프를 자동 실행한다.

## 실행 단계

1. **파일 확인**: `practice-data/A-RnD/` 폴더에서 실적 CSV와 PPR 파일을 읽는다
2. **달성률 분석**: 과제별 Q1 달성률을 계산한다 (소수점 첫째 자리까지)
3. **리스크 과제 분류**: 달성률 70% 미만 과제를 위험/주의로 분류한다
4. **PPR 연결**: 예산 달성률과 PPR 진행 상태를 과제번호 기준으로 통합한다
5. **보고서 생성**: `월간_연구개발비_보고서.md`로 저장한다 (조치 권고 포함)

## 제약 조건
- 달성률은 반드시 소수점 첫째 자리까지 표시 (예: 74.6%)
- 달성률 70% 미만 과제에는 반드시 조치 권고 문장 포함
- 원본 CSV 파일 수정/삭제 금지
- 출력은 표 형식, 한국어

## 출력 예시
```
| 과제번호 | 과제명 | 달성률 | 상태 |
|---------|--------|--------|------|
| RD-008  | ...    | 23.1%  | 🚨 위험 |
```

## 테스트
- `@practice-data/A-RnD/rnd_budget_2026.csv` 파일로 실행 시 달성률 표가 생성되면 성공
- RD-004, RD-008이 주의/위험으로 분류되면 성공
