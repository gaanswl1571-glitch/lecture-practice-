# CLAUDE.md — 조문지 프로젝트 규칙

## Always (항상 따를 규칙)
- 출력은 항상 한국어로
- 표 형식 우선으로 결과 정리
- 연구개발비 분석 시 달성률은 반드시 소수점 첫째 자리까지 표시 (예: 74.6%)
- 주의 과제(달성률 70% 미만)는 항상 조치 권고 문장을 포함
- 분기별 비교가 가능한 경우 전분기 대비 증감도 함께 표시

## Ask (먼저 확인할 것)
- 분석 기준이 불명확하면 먼저 질문
- 파일을 삭제하거나 덮어쓰기 전에 확인

## Never (절대 하지 말 것)
- 원본 CSV 파일 수정/삭제 금지
- 달성률 계산 없이 분석 결과 제출 금지
- 근거 없는 예산 전용/이월 제안 금지
