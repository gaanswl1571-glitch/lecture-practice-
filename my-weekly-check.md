## 📋 주간 점검 프롬프트 6개

### 1. 연구개발비 실적 현황 확인
```
@practice-data/A-RnD/rnd_budget_2026.csv 를 읽고 이번 달 달성률이 70% 미만인 과제를 표로 보여줘
```

### 2. PPR 리스크 과제 점검
```
@practice-data/A-RnD/ppr_report_Q1.csv 에서 리스크 항목이 있는 과제를 찾아 리스크 심각도 순으로 정리해줘
```

### 3. 예산 + PPR 통합 점검
```
@practice-data/A-RnD/rnd_budget_2026.csv @practice-data/A-RnD/ppr_report_Q1.csv 를 함께 보고 이번 주 가장 긴급한 과제 TOP3를 알려줘
```

### 4. 월간 보고서 초안 작성
```
rnd-monthly-review 스킬을 실행해서 이번 달 월간 보고서 초안을 만들어줘
```

### 5. 보고서 요약 (팀장 보고용)
```
@월간_연구개발비_보고서.md 를 읽고 팀장 보고용으로 3줄 요약해줘. 핵심 숫자와 조치 필요 과제만 포함.
```

### 6. 저장 및 올리기
```
저장해줘
올려줘
```
